# Alle Veier 🛣️
### Din norske reisekompis

En mobilvennlig Progressive Web App (PWA) for norske bilturer.
Viser drivstoffpriser, sanntidsvær fra Yr.no og severdigheter i nærheten.

---

## Funksjoner
- ⛽ Drivstoffpriser fra drivstoffpriser.no
- 🌤 Sanntidsvær fra Yr.no med 7-dagersprognose
- 📍 Severdigheter via Wikipedia
- 📱 Kan installeres på iPhone via Safari
- 🌙 Mørkt nordisk design

---

## Publiser på GitHub Pages (gratis)

### Steg 1 — Lag GitHub-konto
1. Gå til github.com
2. Klikk "Sign up" og følg stegene

### Steg 2 — Lag nytt prosjekt
1. Klikk den grønne "New"-knappen
2. Gi den navnet: alleveier
3. Sett til Public
4. Klikk "Create repository"

### Steg 3 — Last opp filer
1. Klikk "uploading an existing file"
2. Dra alle 5 filene inn (index.html, manifest.json, sw.js, icon-192.svg, README.md)
3. Klikk "Commit changes"

### Steg 4 — Slå på GitHub Pages
1. Klikk "Settings" øverst
2. Klikk "Pages" i venstremenyen
3. Under Source velg "Deploy from a branch"
4. Velg "main" og "/ (root)"
5. Klikk "Save"

### Steg 5 — Ferdig!
Etter 1-2 min er appen live på:
https://DITTBRUKERNAVN.github.io/alleveier

---

## Installer på iPhone
1. Åpne lenken i Safari
2. Trykk Del-knappen (pil opp)
3. Velg "Legg til på Hjem-skjerm"
4. Appen er installert!
